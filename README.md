# OpenMarket

A full-stack marketplace application built with React + Node.js/Express + MySQL.

## Features
- Buy & sell products locally
- Real-time chat between buyers and sellers
- Admin dashboard
- Email OTP verification
- Multi-language support
- Dark-themed footer with CMS

## Tech Stack
- **Frontend**: React 19
- **Backend**: Express.js
- **Database**: MySQL
- **Deployment**: Hugging Face Spaces (Docker)
